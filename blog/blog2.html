The Difference Between ID's and Classes

For our assigned blog project, we've been set with the task of explaining the difference between classes and ID's. To a new learner of HTML classes and ID's can initially seem very similar, However, there are some key differences between the two.

I will begin with the similarities: Both ID and class are attributes used in HTML and CSS.
In CSS a ".class" always begins with a period, and an "#id" always begins with a hashtag.
The key difference between the two, is that ID, in best practise, is primarily used to target one element, whereas a class is usually used to target a group of elements.
